## 3. 计划与执行层级

```text
正式设计与里程碑门禁
→ Roadmap 里程碑
→ 任务登记与任务实现计划
→ 逐步施工记录
→ TDD 与验证矩阵
→ Git / 发布证据
```

- Roadmap 回答里程碑、依赖和出口；
- 任务实现计划回答单个任务如何实施、验证和回滚；
- 施工进度版只记录实际状态和证据；
- 已批准里程碑内的普通任务仍须稳定 ID、计划、TDD 和证据；
- 触碰红线必须暂停并重新审批。
